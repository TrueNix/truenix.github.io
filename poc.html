Instructions for Security Team

This page demonstrates a Clickjacking vulnerability. The semi-transparent window below is an iframe loading the vulnerable URL. The green " decoy button is positioned underneath it.

  1. Observe how the " decoy button is aligned behind a clickable element (e.g., a button, link, or toggle) on the framed page.
  2. To see the final attack, edit this file's CSS for #victim-iframe and change opacity from 0.5 to 0.
  3. Refresh the page. The iframe will be invisible.
  4. Clicking the " decoy button will now perform the action on the vulnerable page.
